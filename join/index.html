サークル参加ガイド

2024年 3月 11日 更新

サークル参加条件

参加資格

  • サークル代表者が義務教育を修了していること。
  • 成人向けの頒布物を用意するサークルはサークル代表者が18歳以上であること。
  • 小春六花に関連した作品を当日１点以上配布できること。
    • 頒布できる作品に関しては、各キャラクターの利用ガイドラインをご確認ください。

募集スペース数

イベント全体で直接参加70スペース

参加費用

  • 1スペース (机半分(幅90cm x 奥行き60cm), 椅子2脚, サークル通行証 2枚): 5,000円
  • 2スペース (机全面(幅180cm x 奥行き60cm), 椅子4脚, サークル通行証 4枚): 10,000円

締切

  • 1次締切: 2024年 3月 31日 (日)
  • 2次締切: 2024年 4月中を予定しています
  • サークル情報変更締切: 2024年 6月 3日 (月)

※申し込み状況によっては1次で満了の可能性がございます。
また申し込み多数の場合は、抽選とさせていただく可能性がございます。

頒布物・展示物の注意事項

禁止頒布物

  • 刑法175条に定める「猥褻図画類」を含む物品
  • 古物や法人発行の物品
  • 各キャラクター利用ガイドラインに反した内容を含む物品
  • 手作り既製品を問わず、不特定多数に対する食品（DECOチョコ／企業作成による個別包装されたものは可能）
  • 日本国における各種法令に反する物品

奥付について

頒布する作品の奥付には、下記の内容を 必ず 記載してください。

  • 発行者名(サークル名, ペンネーム可)
  • 発行日
  • 連絡先(メールアドレス, または電話番号)
    • ※SNSなど連絡の際に登録が必要な連絡先はNG
  • 印刷所名(自家印刷やコンビニコピーの場合は省略可)

その他

  • 会場の電源は使用できません。
  • 音楽や動画を頒布する際はオリジナルまたは製作者からの許諾が得られている素材を使用したものに限ります。
  • イベント当日発行の新譜、新刊等は見本誌として回収致します。
  • グッズ等の提出は任意としますが、内容確認のために写真撮影や出力をさせていただく事があります。

注意事項

参加イベントの確認

このページでは「小春六花はヨーグルトが食べたい３」への参加方法をご案内しております。
「AHSession」「ほのかのしおり」の申し込みは、各イベントサイトのサークル参加ガイドから行ってください。

キャンセルについて

  • 原則として、申込後のキャンセルはできません。
  • 抽選、申込内容不備等により落選の場合は、手数料を差し引いて返金させていただきます。
  • 自己都合でのキャンセル・ご返金は出来かねますので予めご了承ください。

中止について

  • 天災や会場側の要請等の不可抗力によりイベントの開催を中止する場合があります。
  • この場合のサークル参加費は手数料を差し引いて返金します。

申し込みの流れ

サークル参加に必要なもの

  • サークルカットの作成
  • EventPortからイベントへの申込、サークルカットの提出
  • サークル参加費の支払い

サークルカットの用意

  • 作成時の注意事項
    • テンプレートをご使用ください。
      • 1スペースの場合 → circlecut_template1sp.png, circlecut_template1sp.psd
      • 2スペースの場合 → circlecut_template2sp.png, circlecut_template2sp.psd
    • 必ずサークル名を入れてください。
    • フルカラーでご作成ください。
    • 縦横のサイズは変更しないでください。
    • サークルカットは、Webサイトやカタログへの掲載を予定しております。
      • 過激・露骨な性描写等はおやめください。不特定多数の方の閲覧が可能なためご配慮をお願いいたします。
  • 提出時のお願い
    • お申し込み時にご提出いただきます。
    • 提出はPNG形式、またはPSD形式でお願いいたします。
    • PSD形式の場合は、レイヤー結合をお願いいたします。

サークルカットテンプレート

申し込み情報の入力

下のボタンから申し込みページを開き、サークル情報の入力を行ってください。

  • 合体申し込みは、双方の申し込み時に必ずその旨をご記入ください。

サークル参加費のお支払い

  • サークル参加費のお支払いはクレジットカード決済のみとなります。
    • クレジットカードをお持ちでない場合は、Vプリカ等のプリペイド式カードをご利用ください。

当日の流れ

  • 後日、お申し込みの際にご入力いただいたメールアドレス宛にサークル参加要項を送付いたしますので、必ずご確認ください。
  • 当日の受付にはQRコードを利用した方法を予定しております。
  • 宅配便による搬入・搬出を予定しております。
  • イベント当日は見本誌回収を行います。予めご了承ください。
  • 追加椅子のご要望は当日運営本部にて申し受けます。詳細は後日送付するサークル参加要項をご確認ください。
