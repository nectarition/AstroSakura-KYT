合同誌企画「小春六花を探しています」

今回のコハタベでは、小春六花ファンの皆さんと合同誌を制作します！

企画概要

  • 「小春六花を見かけた」「小春六花がいた」、いわゆる小春六花の幻覚が見えた場所のイラストや写真とコメントを募集します。
  • イラスト･写真を投稿していただいた方に献本いたします。
  • ページの兼ね合い上、参加人数上限を設けさせていただきます。
  • 合同誌は当日会場で頒布する他、事後BOOTHでの頒布も予定しています。

募集要項

  • 小春六花が写っていそうな風景のイラストや写真に「〇〇に小春六花がいる」「〇〇で小春六花を見つけた」等といったコメントを添えて、下記フォームまでご送付ください。
  • イラストを投稿される方へ
    • イラスト用テンプレート: こちら
    • 風景を基調としたものをご投稿いただければ幸いです。
  • 写真を投稿される方へ
    • 小春六花(夏色花梨, 花隈千冬)を「写真を撮ったらたまたま写っていた」ようにイラストを描き加えていただいても問題ありません。
    • 「みくちゃ」を使用した写真は規約の関係上ご投稿いただけませんので予めご了承ください。
    • 家電量販店等に設置されているパネルの写真ではご参加いただけません。
  • コメントについて
    • コメントは「140文字」以内でご投稿ください。
    • ハッシュタグ(#～～～)の扱いは、Twitter Web Clientのツイート画面で表示されるものと同じものとします。

参加方法

  1. (必須)TwiPlaで参加表明をお願いします。
  2. 締切までに下記フォームからご投稿ください。
